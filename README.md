## Демонстрационный стенд


    requirements:

    docker docker-compose make


Исполльзование:

```shell
# Поднять контейнеры
make up
```
После запуска всех контейнеров будет доступно:
- api по адресу http://172.16.200.4:3000
- веб приложение по адресу http://172.16.200.5:3000


```shell
# Получить список команд
make help
```
