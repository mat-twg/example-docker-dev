## Демонстрационный стенд


    requirements:

    docker docker-compose make


Исполльзование:

```shell
# Поднять контейнеры
make up
```
После запуска всех контейнеров будет доступно:
- веб приложение по адресу http://localhost:8000
- api по адресу http://localhost:8001

```shell
# Получить список команд
make help
```
